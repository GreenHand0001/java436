# 基于Spring Boot + Vue的宠物商城系统 java436

## 项目概述

本项目是一个基于前后端分离的宠物商城系统，适用于课程设计或毕业设计。系统采用模块化设计，具有清晰的目录结构和多个功能模块。

## 技术栈

- 后端：Spring Boot
- 前端：Vue、Vuex
- 数据库：Mybatis
- UI组件库：Element Plus
- 认证机制：JWT
- 数据可视化：Echats
- AI模型：Spring AI
- 存储服务：阿里云OSS

## 功能模块

### 后台管理

- 登录模块
- 用户管理
- 商品管理
- 分类管理
- 服务管理
- 日志管理（使用AOP实现）

### 前台界面

- 登录功能
- 主页展示
- 商品展示
- 购物车与下单
- 黑白主题切换

## 系统角色

- 系统管理员：负责后台管理操作
- 普通用户：浏览商品、下单、查看个人信息

## 运行环境

- 后端：JDK 1.8+，兼容主流数据库
- 前端：Node.js环境，支持现代浏览器
